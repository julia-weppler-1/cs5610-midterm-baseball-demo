# cs5610-midterm-baseball-demo
This is the codebase I generated as part of the CS5610 midterm using ES6+, Express, Bootstrap, and NodeJS Driver for MongoDB

# Baseball Card Demo

## Getting Started

1. **Install dependencies**
   ```
   npm install
   ```

2. **Run the app**
   ```
   npm start
   ```
   The server runs at `http://localhost:3300`.

3. **Set up MongoDB**
   - Connect to your Docker MongoDB container in MongoDB Compass.
   - Create a database called `baseball-cards` and a collection called `cards`.
   - Upload [`baseball-cards.cards.json`](https://github.com/julia-weppler-1/cs5610-midterm-baseball-demo/blob/main/baseball-cards.cards.json) to the `cards` collection.
